# 🧬 Reverse Engineering Fundamentals

Reverse engineering is the process of analyzing a system to understand its structure, behavior, and components.

## 🎯 Goals

- Understand how software works
- Identify vulnerabilities
- Analyze malware
- Bypass protections

---

## 🔍 Types of Analysis

### 1. Static Analysis
Analyzing code without executing it.

Tools:
- Strings
- Disassemblers

---

### 2. Dynamic Analysis
Running the program and observing behavior.

---

## 🧠 Key Concepts

- Assembly language
- Binaries (ELF, PE)
- Memory
- Processes

---

## 🛠️ Common Tools

- Ghidra
- IDA Free
- Radare2
- x64dbg

---

## ⚠️ Important
Reverse engineering is essential for malware analysis and vulnerability research.
